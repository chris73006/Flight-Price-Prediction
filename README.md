Flight Price Prediction and Analysis

Project Overview

This project focuses on predicting airline ticket prices using Machine Learning and analyzing flight pricing trends through an interactive Power BI dashboard.

The objective is to identify key factors affecting flight prices and build a predictive model capable of estimating ticket costs based on historical flight data.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Power BI

Dataset Features

The dataset contains information such as:

- Airline
- Flight
- Source City
- Destination City
- Departure Time
- Arrival Time
- Stops
- Travel Class
- Duration
- Days Left
- Price

Machine Learning Model

Random Forest Regressor

Steps performed:

1. Data Loading and Cleaning
2. Feature Encoding using Label Encoding
3. Train-Test Split
4. Model Training
5. Price Prediction
6. Model Evaluation

Model Performance

- MAE (Mean Absolute Error): 862.97
- RMSE (Root Mean Squared Error): 2319.16
- R² Score: 0.9896

The model achieved high prediction accuracy and effectively captured the relationship between flight attributes and ticket prices.

Project Outcome

This project demonstrates how Machine Learning and Business Intelligence tools can be combined to analyze airline pricing trends and generate accurate flight price predictions.
